📋 MENÚ INTERACTIVO (python sac_final.py)
════════════════════════════════════════════════════════════════════════════════

  1. ✅ Verificación de sistema y salud
     → Chequea que todo esté configurado correctamente

  2. 🔍 Validar Orden de Compra (OC)
     → Valida una OC individual contra distribuciones

  3. 📊 Generar Reporte Diario de Planning
     → Crea reporte automático con estado actual

  4. 📧 Enviar Alerta Crítica
     → Envía email de alerta inmediatamente

  5. 🔄 Validar Múltiples OCs (Lote)
     → Procesa archivo CSV con múltiples OCs

  6. 📦 Programa de Recepción
     → Genera programa de recepción automático

  7. 🚨 Monitor en Tiempo Real
     → Monitoreo continuo con alertas automáticas

  8. 📈 Generar Reporte de Errores
     → Reporte consolidado de errores detectados

  9. 🛠️ Configuración y Utilidades
     → Opciones de configuración y herramientas

  10. 📚 Documentación y Ayuda
      → Acceso a documentación y guías

⚡ COMANDOS CLI PRINCIPALES
════════════════════════════════════════════════════════════════════════════════

Validar una OC:
  $ python sac_final.py --validate-oc 750384123456
  Resultado: Reporte completo de validación

Generar reporte diario:
  $ python sac_final.py --report-daily
  Resultado: Reporte Excel en output/resultados/

Monitor en tiempo real:
  $ python sac_final.py --monitor
  Resultado: Monitoreo continuo (Ctrl+C para detener)

Verificar salud:
  $ python sac_final.py --health
  Resultado: Diagnóstico rápido del sistema

Ver configuración:
  $ python sac_final.py --config
  Resultado: Configuración actual del sistema

  $ python health_check.py --detailed

  $ python setup_env_seguro.py
  (Reconfigura credenciales de email)

Problema: "Permiso denegado"
  $ chmod -R 755 output/
