# Resumen del Proyecto SAC

## Sistema de Automatización de Consultas
**CEDIS Cancún 427 - Tiendas Chedraui S.A. de C.V.**

### Objetivo
Automatizar la validación y monitoreo de órdenes de compra (OC) para el departamento de Planning, integrándose con Manhattan WMS (DB2).

### Características Principales
- ✅ Conexión directa a IBM DB2 (Manhattan WMS)
- ✅ Validación automática de OC vs Distribuciones
- ✅ Detección de errores críticos (distribuciones excedentes)
- ✅ Generación de reportes Excel profesionales
- ✅ Notificaciones por email, Telegram y WhatsApp
- ✅ Dashboard web para visualización
- ✅ Modo daemon para ejecución continua
